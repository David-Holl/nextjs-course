# Next.js & React Project

## Credits

This project is based on the Udemy course [`Next.js & React - Der komplette Kurs`](https://www.udemy.com/course/nextjs-react/) by Fabian Heihoff. Full credit goes to him for the course content and guidance.

## Purpose

This repository serves as a hands-on learning project for building modern web applications with React and Next.js, covering everything from the basics to a full-scale production-ready web project.

## Content

The implementation follows the core concepts of the course but may include custom adjustments and best practices in project organization and styling.

### React Grundlagen

- **States & Redux:** Grundlagen der State-Verwaltung und globaler State mit Redux.
- **Styling:** Design-Anpassungen mit CSS und SASS.
- **Komponenten:** Aufbau modularer UI-Elemente.

### Next.js Grundlagen

- **Setup:** Installation und Einrichtung von Next.js Projekten.
- **Routing:** Navigation und File-based Routing.
- **API Endpoints:** Erstellung und Nutzung von Serverless Functions / API-Routen.
- **Deployment:** Kompilierung und Veröffentlichung der Anwendung im Web.

### Next.js Webprojekt

- **Full-stack Development:** Entwicklung einer realen, funktionsfähigen Webanwendung.
- **Datenbanken:** Integration von lokalen und Cloud-Datenbanken (z. B. MongoDB/PostgreSQL).
- **Payment:** Implementierung von PayPal zur Abwicklung echter Zahlungen.
- **Animationen:** Einsatz von Transitions für flüssige Benutzererfahrungen.
- **NPM Pakete:** Verwendung zahlreicher JavaScript-Bibliotheken zur Funktionserweiterung.
